## Описание работы
**Цель работы**: освоить современные технологии CSS-разметки — Grid.
Адаптивная верстка — это подход к созданию веб-страниц, при котором дизайн
и контент автоматически подстраиваются под размер экрана устройства (компьютер,
планшет, смартфон).
**Grid** — двумерная система компоновки для создания сложных сеток.
